# Changelog

All notable changes to this project will be documented in this file.

## [0.8.0] - 2026-04-19

### Added
- New `Ignore property` prefix setup. Basically if you add the matching prefix from the settings which is `#` by default, to the start of any Notion property name, the setup will now ignore that property and not attempt to parse it. 
- Added contribution & changelog flows to the project.

### Changed
- Version validation updated to the v2 setup.
- Updated the asset licence to GPL3, matching my other works.
